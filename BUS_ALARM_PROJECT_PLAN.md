# Bus Alarm 開發項目計劃

## 項目目標
開發一個公交提醒系統，讓用戶可以設定特定路線和站點，當公車即將到達時接收提醒通知。

## 已完成的工作
1. 修復香港交通API連接問題
   - 修正了"inbound"/"outbound"方向參數
   - 實現了動態stop ID獲取功能

## 待開發功能
1. 用戶界面設計 (設定提醒)
2. 公交到站監控系統
3. 提醒通知機制
4. 用戶設定管理

## 技術架構
- 後端: Node.js + Express
- 數據庫: MongoDB
- API: 香港運輸署實時數據API
- 通知: Telegram/手機推送

## 開發步驟
1. 設計數據模型
2. 建立API接口
3. 實現監控邏輯
4. 開發用戶交互功能
5. 測試和部署
